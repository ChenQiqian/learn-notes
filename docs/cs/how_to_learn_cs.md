# 如何系统学习计算机科学？

以 thu 计算机专业的现状魔改...

每年三个学期，每个学期 12 周课时 + 2 周考试；学期间 4 周假期。

> 1 credit = 12 hrs 

## 数学课组

必修：

+ 微积分(1)(2)
+ 线性代数(1)(2)
+ 计算机数学基础
+ 概率论

选修：

+ 微积分高级专题
+ 线性代数高级专题
+ 计算机数学高级专题(抽象代数)
+ 数理统计与随机过程

## 计算机基础知识

历史：

+ 计算机科学历史
+ 计算机工程历史

基础编程:

+ 程序语言理论与实践(1)(2)
+ 数据结构与算法

编程实践：

+ 软件工程
